Centralized Automated Solution for Price Estimate and Reasonability

Project Overview

Centralized Automated Solution for Price Estimate and Reasonability is a web-based application developed to automate product price estimation and evaluate whether a given price is reasonable based on historical data. The system helps organizations reduce manual effort, improve pricing accuracy, and maintain a centralized database for price management.

The project is developed using Python Flask as the backend framework and MySQL as the database management system.
